LawVu UI Tech Test

The LawVu UI Tech Test is built using Angular 20 with standalone components and modern control flow.
Below are all the instructions for the frontend requirements.

Running the Project

The solution uses .NET Aspire for orchestration and runs everything in Docker containers. No Node.js installation required!

  1. Open Docker Desktop
  2. Open the LawVuTechTest.sln file in Visual Studio
  3. Set TechTest.AppHost as the startup project
  4. Run the solution with F5 or Ctrl+F5
  5. The Aspire dashboard will open showing all services
  6. Access the Angular app at http://localhost:4200
  7. Access the WebAPI at http://localhost:9091

The Angular WebApp runs in a Docker container with live reload - edit files directly and see changes automatically!
